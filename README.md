# Recursos Humanos (RH)

Este repositório reúne atividades, estudos de caso, documentos, projetos e materiais desenvolvidos pelas turmas do curso de Recursos Humanos do Senac Montenegro.

Você encontrará:

- Exercícios e atividades práticas  
- Modelos de documentos de RH  
- Estudos e análises  
- Materiais complementares

Um ambiente organizado para apoiar o aprendizado e a prática profissional.
